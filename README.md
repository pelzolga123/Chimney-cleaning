* HTML5
* Javascript

* Bootstrap


## Development

* Clone project
```
git clone https://github.com/pelzolga123/Chimney-cleaning.git
```
* Install dependencies
```
npm install
```
* Run script
```
npm run build
```
Then open index.html in the browser.
